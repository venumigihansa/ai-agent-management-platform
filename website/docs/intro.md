# Welcome to AI Agent Management Platform

The AI Agent Management Platform (AMP) is a comprehensive solution for building, deploying, and managing AI agents at scale.

## Quick Links

- [Quick Start Guide](./getting-started/quick-start.md)
- [Installation Guide](./getting-started/installation.md)

## Features

- **Agent Lifecycle Management**: Build, deploy, and monitor AI agents
- **Observability**: Built-in tracing and monitoring with OpenTelemetry
- **Multi-language Support**: Python instrumentation with more languages coming soon
- **Scalable Architecture**: Kubernetes-native design for cloud deployments
